# choo-web server

SSR'd choo frontend.


## SYNOPSIS

choo-web [flags]


## OPTIONS

### -p PORT, --port PORT (required, default=8080)

The port that this web server listens on.

### --dev (default=false)

Should we start as a development server?

### --graphql-uri (required, default=http://localhost:5000)

Location of the GraphQL server.
